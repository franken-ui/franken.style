Frankternity is the newest library added to the Frankenstyle Project. It is a collection of beautifully designed components intended to attract user attention, primarily for marketing use. It is a port of Aceternity UI.

Each Frankternity component has its own installation instructions, which can be found in the corresponding component documentation.
