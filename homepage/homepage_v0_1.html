Frankenstyle

No-Build, Value-Driven CSS Framework

A utility-first CSS framework that requires no build process. Just drop in the CDN and start styling with value-driven utilities.

Features

Why Frankenstyle?

Built around simple, powerful principles for modern web development.

No Build Process

Everything is pre-compiled and ready to use via CDN. No bundler, no watchers, no config files.

Value-Driven

Generic utility names paired with CSS variables you define. No memorizing arbitrary scales.

Runtime States

Hover, focus, and other states are generated at runtime, keeping your CSS bundle small.

Responsive Design

Familiar breakpoint prefixes (sm:, md:, lg:) for building responsive interfaces.

Theme-Aware

Built-in dark mode support with the dark: prefix for seamless theming.

Explicit Control

Intentional styling with utilities and values that always go together.

Getting Started

Installation

Get started in seconds with CDN or bundler.

Recommended

CDN Installation

Perfect for beginners. Just add the stylesheet to your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/frankenstyle@0.1.0-next.11/dist/css/frankenstyle-kit.min.css">

No build step required. Start using utilities immediately.

Bundler Installation

Install via npm for integration with build tools:

npm install frankenstyle

Then import in your CSS:

@import 'frankenstyle/dist/css/frankenstyle-kit.min.css';
Quick Start

See Frankenstyle in Action

Explore the unique value-driven approach.

Value-Driven Utilities 
<!-- Use generic utility classes with custom property values -->
<div 
  class="m sm:m md:m" 
  style="--m: 4; --sm-m: 8; --md-m: 16"
>
  Responsive margins
</div>
Interactive States 
<!-- Add hover and focus states with the data-uk attribute -->
<button
  class="bg bg:hover px py rounded-md"
  style="
    --bg: var(--uk-primary);
    --bg-hover: oklch(50% 0.245 262.881);
    --px: 5;
    --py: 2.5;
  "
  data-uk
>
  Hover me
</button>
