CSS utilities without the build step.

Frankenstyle is a value-driven CSS framework. No bundler required—just include the stylesheet and start building with utilities that make sense.

Get Started View on GitHub
Features

The right abstractions, nothing more.

No Build Process

Pre-compiled and ready via CDN. No bundlers, no watchers, no config.

Value-Driven

Generic utilities paired with CSS variables you control. No arbitrary scales.

Runtime States

Hover, focus, and active states generated at runtime. Smaller bundles.

Responsive

Breakpoint prefixes (sm:, md:, lg:) for layouts that adapt to any screen.

Dark Mode

Built-in theming with the dark: prefix. Seamless light and dark support.

Explicit

No magic. Utilities and values always go together intentionally.

Usage

Write styles that read like intent.

example.html
<!-- Value-driven responsive margins --> <div class="m sm:m md:m" style="--m: 4; --sm-m: 8; --md-m: 16" > Content with responsive margins </div>
Installation

Start building in seconds.

Recommended

CDN

Add to your HTML head:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/frankenstyle@0.1.0-next.11/dist/css/frankenstyle-kit.min.css">

npm

Install via package manager:

npm install frankenstyle

Ready to build?

One stylesheet. Zero configuration. Complete control over your styles.
