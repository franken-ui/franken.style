Features

Built different.

No Build Process

Everything is pre-compiled and ready via CDN. No bundlers, no watchers, no config files. Just include and build.

Value-Driven

Generic utilities with CSS variables you control. No memorizing arbitrary scales.

Runtime States

Hover, focus, and more generated at runtime for lean CSS.

Fully Responsive

Breakpoint prefixes (sm:, md:, lg:) for any screen size.

Dark Mode

Built-in theming with the dark: prefix. Zero config.

Code

See how it works.

<!-- Value-driven responsive utilities --> <div class="m sm:m md:m lg:m" style="--m: 2; --sm-m: 4; --md-m: 8; --lg-m: 16" > Responsive margins </div>
Install

Get started in seconds.

Recommended

CDN

Add this to your HTML <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/frankenstyle@0.1.0-next.11/dist/css/frankenstyle-kit.min.css">

npm

Install with your package manager:

npm install frankenstyle

Ready to build?

One stylesheet. Zero config. Maximum control.
