CSS utilities without the build step.

Frankenstyle is a value-driven CSS framework. No bundler required—include one stylesheet and build interfaces with utilities that make sense.

Features

The right abstractions. Nothing more.

01

No Build Process

Pre-compiled and ready via CDN. No bundlers, watchers, or configuration files.

02

Value-Driven

Generic utilities with CSS variables you control. No arbitrary scales to memorize.

03

Runtime States

Hover, focus, and active states generated at runtime for smaller bundles.

04

Responsive

Breakpoint prefixes (sm:, md:, lg:) for layouts that adapt to any screen.

05

Dark Mode

Built-in theming with dark: prefix. Seamless light and dark mode support.

06

Explicit

No magic. Utilities and values always go together intentionally.

Installation

Start building in seconds.

Recommended

CDN

Add this to your HTML <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/frankenstyle@0.1.0-next.11/dist/css/frankenstyle-kit.min.css">

npm

Install with your package manager:

npm install frankenstyle

Ready to build?

One stylesheet. Zero config. Complete control over your styles.
